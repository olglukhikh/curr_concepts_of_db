# Лабораторная №1
## **Вариант №5**

База данных "Банковские вклады" должна хранить следующую информацию:
- Фамилия вкладчика.
- Имя вкладчика.
- Отчество вкладчика.
- Номер счета в банке.
- Дата посещения.
- Сумма взноса.
- Сумма выдачи.
- Серия и номер паспорта вкладчика.
- Домашний адрес вкладчика.
